# What Language Is That?
A resource pack for Minecraft Java Edition that changes all Standard Galactic Alphabet (Enchanting Table language) into regular English!
Now supports versions as far back as JE 1.0.0!

[Check out the WLIT Wiki for more information](https://github.com/Ennismore0/What-Language-Is-That/wiki/What-Language-Is-That%3F-Wiki)
